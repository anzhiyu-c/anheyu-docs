## 6. 我想用 CDN 加速我的网站

平台自带高速缓存，后续加载会命中高速缓存，使用 CDN 需要注意，每次更新版本都需要刷新 CDN 缓存，否则可能更新不会生效，建议不缓存 html 文件。

## 7. 我想自定义代码

进入 `系统设置` -> `页面设置`，可以添加自定义 HTML、CSS 和 JavaScript 代码。

### 支持的代码类型

| 类型            | 说明                   | 使用场景              |
| --------------- | ---------------------- | --------------------- |
| 自定义头部 HTML | 插入到 `<head>` 内     | 添加 meta、外部样式等 |
| 自定义底部 HTML | 插入到 `</body>` 前    | 添加统计代码、插件等  |
| 自定义 CSS      | 全局样式               | 修改主题颜色、字体等  |
| 自定义 JS       | 全局脚本               | 添加自定义功能        |
| 文章顶部 HTML   | 插入到文章内容区域顶部 | 添加提示、赞助信息等  |
| 文章底部 HTML   | 插入到文章内容区域底部 | 添加版权声明、推荐等  |

### 常用示例

**添加统计代码（自定义底部 HTML）：**

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=YOUR_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag() {
    dataLayer.push(arguments);
  }
  gtag("js", new Date());
  gtag("config", "YOUR_ID");
</script>
```

**修改主题色（自定义 CSS）：**

```css
:root {
  --theme-color: #ff6b6b;
}
```

**添加版权声明（文章底部 HTML）：**

```html
<div style="padding: 15px; background: #f5f5f5; border-radius: 5px; margin-top: 20px;">
  <p>
    <strong>版权声明：</strong>
    本文采用
    <a href="https://creativecommons.org/licenses/by-nc-sa/4.0/" target="_blank">CC BY-NC-SA 4.0</a>
    许可协议
  </p>
</div>
```

### 注意事项

- ✅ 代码保存后立即生效，刷新页面即可看到
- ⚠️ 确保代码来源可信，不要暴露敏感信息
- ⚠️ 外部脚本使用 `async` 或 `defer` 异步加载
- ⚠️ 修改前建议备份代码
