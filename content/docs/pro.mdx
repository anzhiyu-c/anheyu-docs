Anheyu Pro 是 Anheyu App 的功能增强版本，在基础版本上扩展了更多专业特性。

在 [安和鱼官网](https://anheyu.com) 注册并登录后可以添加根域名授权。

## 功能概览

| 功能分类     | 功能模块                                   |
| ------------ | ------------------------------------------ |
| **AI 能力**  | AI 摘要、AI 播客、AI 写作                  |
| **内容保护** | 付费内容、密码保护、全文隐藏、登录可见     |
| **支付系统** | 支付宝/微信支付、易支付、虎皮椒、订单管理  |
| **社交功能** | 第三方登录、朋友圈、说说（即刻）、打赏名单 |
| **SEO 优化** | 百度/Bing/Google 自动提交                  |
| **其他功能** | 文件分享、微信公众号登录、数据迁移         |

## AI 功能

### AI 摘要

支持 GLM、OpenAI、Claude 等多个 AI 模型，自动从文章内容中提取关键信息生成摘要。

### AI 播客

![AI 播客](https://upload-bbs.miyoushe.com/upload/2025/11/05/125766904/9d759897f65eb5e7d59d54e54806e671_970332254315767898.png)

将文章内容转换为音频播客，支持多种 TTS 引擎和声音选择，可在文章页面直接播放。

### AI 写作

提供 AI 辅助写作功能，支持内容续写、润色、改写等操作，提升写作效率。

## 内容保护

### 付费内容

![付费内容](https://upload-bbs.miyoushe.com/upload/2025/09/15/125766904/525985d697e736ba95fb847590935931_480370356362226895.png?x-oss-process=image/format,avif)

- 支持整篇或分段付费
- 设置原价和活动价
- 支持游客邮箱购买
- 实时销售统计

### 密码保护

![密码保护](https://upload-bbs.miyoushe.com/upload/2025/10/03/125766904/85c62c3d670149a7b208a758ff74ced1_5838447315971595190.png?x-oss-process=image/format,avif)

为文章特定段落设置密码保护，支持密码提示、尝试次数限制和有效期设置。

### 全文隐藏

支持隐藏文章全文内容，用户需满足特定条件（如评论、关注）才能查看。

### 登录可见

设置文章内容仅登录用户可见，引导用户注册登录。

## 支付系统

![支付配置](https://upload-bbs.miyoushe.com/upload/2025/09/25/125766904/7fcc31f0f0e232163a4cd57eb0f6cf13_7907934964932495514.png?x-oss-process=image/format,avif)

| 支付通道     | 说明               |
| ------------ | ------------------ |
| 支付宝当面付 | 原生支付宝扫码支付 |
| 微信支付 V3  | 原生微信扫码支付   |
| 易支付       | 聚合支付通道       |
| 虎皮椒 V3    | 聚合支付通道       |

> 原生通道与聚合通道不可同时启用

详细配置请参考 [支付配置指南](/docs/payment-configuration)。

## 社交功能

### 第三方登录

![第三方登录](https://upload-bbs.miyoushe.com/upload/2025/10/22/125766904/ff1a980516fdf63bf8cbb5aec3ceba32_4487066571638655429.png?x-oss-process=image/format,avif)

支持 QQ、微信、GitHub、自定义 OIDC 等多平台登录，自动同步头像和昵称。

### 朋友圈

![朋友圈](https://upload-bbs.miyoushe.com/upload/2025/10/22/125766904/02ffc873ea09e97b0f664eb7ab1132d8_6668867194887043606.png?x-oss-process=image/format,avif)

自动聚合友链博客的最新文章，支持 RSS 自动发现、定时抓取和新文章通知。

### 说说（即刻）

![说说即刻](https://upload-bbs.miyoushe.com/upload/2025/10/22/125766904/b7e9d81ee56050e973411dcb8d2a86b3_5268642522317200594.png?x-oss-process=image/format,avif)

快速发布动态内容，支持文字、图片、音频、地理位置等多种类型。

### 打赏名单

![打赏名单](https://upload-bbs.miyoushe.com/upload/2025/12/27/125766904/b7cb05ad817570f14ad0c8fcb602aa63_8070541148429228847.png?x-oss-process=image/format,avif)

展示打赏用户列表，支持自定义展示样式和排序方式。

## SEO 优化

![SEO配置](https://upload-bbs.miyoushe.com/upload/2025/12/27/125766904/3ea52b6168af13015a05882036893c50_1587048903550044948.png?x-oss-process=image/format,avif)

文章发布/更新时自动提交到搜索引擎：

- **百度**：使用百度站长平台 API
- **Bing**：使用 Bing Webmaster Tools API
- **Google**：使用 Google Indexing API

支持批量提交、重试机制和异步处理。

## 其他功能

### 文件分享

创建文件分享链接，支持密码保护、有效期和下载次数限制。

![文件分享](https://upload-bbs.miyoushe.com/upload/2025/12/27/125766904/ab3f44ac150c910c9d6433e4c38127a0_5680340213490256885.png?x-oss-process=image/format,avif)

### 微信公众号登录

支持微信公众号扫码登录，适用于无法使用微信开放平台的场景。

### 数据迁移

从其他博客系统（如 Hexo、WordPress）迁移数据到 Anheyu。
