## 关于 Anheyu Pro

Anheyu Pro (`anheyu-pro`) 是 Anheyu App (`anheyu-app`) 的功能增强扩展版本。它在保留 Anheyu App 所有基础功能的基础上，提供了更多专业和高级的特性。

您可以选择赞助 Anheyu 的开发，作为回报，您可以获得功能加强的 Anheyu Pro 版本。

## 核心功能

Pro 版本在基础版本之上扩展了以下专业功能：

### 1. AI 智能摘要

- **多模型支持**：支持 GLM（智谱清言）、OpenAI GPT、Claude 等多个 AI 模型
- **智能提取**：自动从文章 HTML 内容中提取文本并生成摘要

### 2. 付费内容系统

- **灵活定价**：支持文章整体或分段付费，可设置原价和活动价
- **内容保护**：付费内容在数据库中独立存储，确保安全性
- **预览功能**：支持为付费内容设置预览部分，吸引用户购买
- **销售统计**：实时统计销量、查看次数、总收入等数据
- **多货币支持**：支持 CNY 等多种货币单位
- **内容管理**：支持内容状态管理（草稿、发布、下架、过期）

### 3. 密码保护内容

- **段落级保护**：支持对文章中的特定段落设置密码保护
- **安全加密**：使用 bcrypt 哈希算法加密密码
- **友好提示**：可自定义标题、密码提示和占位符文本
- **访问控制**：支持最大尝试次数限制和密码过期时间
- **访问统计**：记录访问次数和最后访问时间

### 4. 支付系统

- **支付宝**：支持支付宝当面付接口
- **微信支付**：支持微信官方支付 V3 接口
- **完整流程**：订单创建、支付查询、回调处理、订单管理
- **安全可靠**：支持订单加密、回调验证、重复支付检测
- **游客支付**：支持游客通过邮箱进行购买

### 5. 用户购买管理

- **购买记录**：完整记录用户购买历史
- **访问令牌**：为每个购买生成唯一访问令牌
- **访问控制**：支持设置访问次数限制和有效期
- **多端同步**：通过邮箱和令牌实现多设备访问

### 6. 存储策略扩展

- **米游社同步**：支持图片自动同步到米游社图床
- **OSS 处理**：支持阿里云 OSS 图片处理样式（如格式转换、压缩等）
- **链接管理**：自动管理原图和米游社链接的映射关系

### 7. SEO 搜索引擎自动提交

- **多引擎支持**：支持百度、Bing、Google 三大搜索引擎
- **自动提交**：文章发布/更新时自动提交到已启用的搜索引擎
- **配置管理**：可视化配置各搜索引擎的 API 参数
- **测试连接**：一键测试搜索引擎配置是否正确
- **批量提交**：支持手动批量提交多个 URL
- **重试机制**：可配置重试次数和间隔时间
- **异步处理**：后台异步提交，不影响文章发布速度

### 8. 第三方登录系统

- **多平台支持**：支持 QQ、微信、GitHub 等主流平台登录
- **OAuth 2.0**：基于标准 OAuth 2.0 协议实现
- **账号绑定**：支持多个第三方账号绑定到同一用户
- **自动注册**：首次登录自动创建用户账号
- **信息同步**：自动同步头像、昵称等基础信息
- **安全可靠**：支持状态验证和回调验证

### 9. 朋友圈系统

- **RSS 聚合**：自动抓取友链博客的最新文章
- **智能发现**：自动发现和解析友链的 RSS 订阅源
- **实时更新**：定时抓取更新，保持内容新鲜度
- **缓存优化**：智能缓存机制，提升访问速度
- **统计分析**：提供友链活跃度和文章数量统计
- **页面配置**：可视化配置朋友圈页面样式和内容
- **通知推送**：支持 Webhook 通知新文章更新

### 10. 说说（即刻）系统

- **动态发布**：支持发布文字、图片、链接等多种类型内容
- **多媒体支持**：支持图片上传和音频播放器配置
- **状态管理**：支持草稿、发布等多种状态
- **自定义发布时间**：支持设置自定义的发布时间
- **排序管理**：支持自定义排序顺序
- **地址标记**：支持添加地理位置信息
- **来源标记**：支持标记内容来源

## 功能详解

### AI 摘要

![AI 摘要(PRO)](https://upload-bbs.miyoushe.com/upload/2025/09/15/125766904/e942964f43c1da5ec045e445dd5ded78_5585331757237990077.png?x-oss-process=image/format,avif)

在编辑文章时，点击 `AI生成` 按钮即可自动生成文章摘要。系统会智能分析文章内容，提取关键信息并生成简洁的摘要。

### 付费文章内容

![](https://upload-bbs.miyoushe.com/upload/2025/09/15/125766904/525985d697e736ba95fb847590935931_480370356362226895.png?x-oss-process=image/format,avif)

创建付费文章时，你可以：

- 设置整篇文章的价格
- 支持分段付费，不同段落设置不同价格
- 设置原价和活动价，展示优惠力度
- 添加付费内容预览，吸引用户购买
- 查看实时销售数据和收入统计

付费内容会在独立的数据库表中加密存储，只有购买后的用户才能通过访问令牌查看完整内容。

### 支付配置

![支付(PRO)](https://upload-bbs.miyoushe.com/upload/2025/09/25/125766904/7fcc31f0f0e232163a4cd57eb0f6cf13_7907934964932495514.png?x-oss-process=image/format,avif)

在系统后台的支付配置页面，填写对应的支付参数：

#### 支付宝配置

- **App ID**：支付宝开放平台应用 ID
- **私钥**：应用私钥
- **公钥**：支付宝公钥
- **回调地址**：支付成功后的通知地址

#### 微信支付配置

- **商户号**：微信支付商户号
- **API 密钥**：API v3 密钥
- **证书**：商户证书文件
- **回调地址**：支付成功后的通知地址

配置完成后，启用对应的支付方式即可开始使用。系统会自动处理订单创建、支付查询、回调验证等流程。

### 密码保护内容

![](https://upload-bbs.miyoushe.com/upload/2025/10/03/125766904/85c62c3d670149a7b208a758ff74ced1_5838447315971595190.png?x-oss-process=image/format,avif)

在创建文章时，你可以为特定段落设置密码保护：

- 选择需要保护的内容段落
- 设置访问密码（系统会使用 bcrypt 加密存储）
- 可选：添加密码提示信息
- 可选：设置最大尝试次数和密码有效期

用户访问文章时，需要输入正确的密码才能查看被保护的内容。密码保护的内容同样在独立数据库中加密存储。

### SEO 搜索引擎自动提交

![SEO配置](https://upload-bbs.miyoushe.com/upload/2025/10/22/125766904/a04caeca0f5fea44dd9c090bf8eb8e63_2618184010811007218.png?x-oss-process=image/format,avif)

Pro 版本提供了强大的 SEO 搜索引擎自动提交功能，帮助您的文章更快被搜索引擎收录：

#### 支持的搜索引擎

**百度搜索**

- 使用百度站长平台 API
- 需要在百度搜索资源平台验证站点
- 获取推送 Token 进行配置

**Bing 搜索**

- 使用 Bing Webmaster Tools API
- 需要在 Bing 站长工具验证站点
- 获取 API 密钥进行配置

**Google 搜索**

- 使用 Google Indexing API
- 需要配置 Google Cloud 服务账号
- 支持 OAuth 2.0 认证

#### 功能特性

- **自动提交**：文章发布或更新时自动提交到已启用的搜索引擎
- **测试连接**：配置完成后可一键测试连接是否正常
- **批量提交**：支持手动选择多个 URL 进行批量提交
- **重试机制**：提交失败时自动重试，可配置重试次数和间隔
- **异步处理**：后台异步提交，不影响文章发布速度
- **详细日志**：提供详细的提交日志和结果反馈

#### 配置步骤

1. 在系统管理 → 设置管理 → 前台设置 → SEO 配置 中进行配置
2. 填写对应搜索引擎的站点信息和 API 密钥
3. 点击"测试连接"验证配置是否正确
4. 启用"自动提交"开关
5. 保存配置即可开始使用

### 第三方登录

![第三方登录](https://upload-bbs.miyoushe.com/upload/2025/10/22/125766904/ff1a980516fdf63bf8cbb5aec3ceba32_4487066571638655429.png?x-oss-process=image/format,avif)

Pro 版本支持多种第三方平台登录，为用户提供便捷的登录体验：

#### 支持的平台

- **QQ 登录**：基于 QQ 互联平台
- **微信登录**：支持微信开放平台
- **GitHub 登录**：支持 GitHub OAuth
- **自定义 OIDC**：支持标准 OpenID Connect 协议

#### 功能特性

- **一键登录**：用户无需注册，直接使用第三方账号登录
- **账号绑定**：已有用户可绑定多个第三方账号
- **信息同步**：自动同步头像、昵称等基础信息
- **安全验证**：支持状态码验证和回调地址验证
- **自动注册**：首次登录自动创建用户账号

#### 配置方法

1. 在各第三方平台创建应用并获取 Client ID 和 Secret
2. 在系统管理 → 第三方登录配置中填写相关参数
3. 设置回调地址和授权范围
4. 启用对应的登录方式
5. 用户即可在登录页面看到第三方登录选项

#### 用户体验

- 登录页面显示第三方登录按钮
- 点击后跳转到对应平台授权页面
- 授权成功后自动返回并完成登录
- 支持账号绑定和解绑操作

### 朋友圈系统

![朋友圈](https://upload-bbs.miyoushe.com/upload/2025/10/22/125766904/02ffc873ea09e97b0f664eb7ab1132d8_6668867194887043606.png?x-oss-process=image/format,avif)

Pro 版本提供了强大的朋友圈系统，可以自动聚合友链博客的最新文章，为访客提供丰富的内容发现体验：

#### 核心功能

**RSS 自动发现**

- 智能识别友链网站的 RSS 订阅源
- 支持多种 RSS 格式（RSS 2.0、Atom、JSON Feed）
- 自动解析文章标题、链接、摘要、发布时间等信息

**定时抓取更新**

- 可配置抓取频率和时间
- 支持增量更新，只抓取新发布的文章
- 智能去重，避免重复内容

**缓存与性能**

- 多层缓存机制，提升访问速度
- 支持缓存预热和手动清理
- 优化数据库查询，减少服务器负载

#### 管理功能

**友链管理**

- 自动关联已审核的友情链接
- 支持手动触发特定友链的抓取
- 提供友链活跃度统计

**内容管理**

- 查看所有抓取的文章列表
- 支持按友链、时间等条件筛选
- 提供文章统计和分析数据

**系统监控**

- 实时查看抓取状态和进度
- 支持手动触发 RSS 抓取任务
- 支持清理朋友圈缓存
- 提供抓取统计信息

#### 页面配置

**外观设置**

- 自定义页面标题和副标题
- 配置顶部背景图片
- 配置按钮文字和跳转链接

**显示配置**

- 设置最大抓取条目数（maxItems）
- 配置页面显示限制（displayLimit）
- 设置缓存持续时间（cacheDuration）
- 配置 RSS 抓取超时时间

**通知设置**

- 支持管理员邮件通知
- 支持 Server 酱 邮件通知
- 支持 Pushoo 多渠道推送（Webhook、Bark 等）
- 可配置通知模板和推送内容

#### 使用场景

- **内容聚合**：为访客提供友链最新动态
- **社区建设**：增强博客间的互动和联系
- **内容发现**：帮助用户发现优质的友链内容
- **活跃度监控**：了解友链的更新频率和活跃程度

#### 配置步骤

1. 在系统管理 → 设置管理 → 前台设置 → 朋友圈配置 中启用功能
2. 配置页面基本信息（标题、副标题、背景图等）
3. 设置抓取参数（抓取间隔、最大条目数、显示限制等）
4. 配置通知方式（邮件、Server 酱、Pushoo 等）
5. 添加友情链接，系统会自动发现 RSS 源并定时抓取

#### 技术特点

- **高性能**：采用异步抓取和智能缓存
- **高可靠**：支持重试机制和错误恢复
- **易扩展**：模块化设计，便于功能扩展
- **用户友好**：提供详细的配置界面和状态反馈
